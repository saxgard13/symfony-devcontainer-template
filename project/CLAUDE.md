# Projet — Instructions globales Claude Code

## Structure
```
project/
├── backend/    # Application Symfony (PHP)
└── frontend/   # Application frontend (JS/TS)
```

## Backend
- Entrypoint : `project/backend/`
- Instructions détaillées : `project/backend/CLAUDE.md` (à créer)

## Frontend
- Entrypoint : `project/frontend/`
- Instructions détaillées : `project/frontend/CLAUDE.md` (à créer)

## Conventions générales
- Ne jamais committer automatiquement
- Pas d'over-engineering : solution la plus simple qui répond au besoin
- Signaler les problèmes hors scope sans les modifier
- Ne pas écrire de tests sauf si explicitement demandé
